## Requirements
- MongoDB
- Nodejs >= 10.0.0

## Installation

- Copy project on your local machine.
- Run npm install.
- Set your mongodb database (src/config/db).
- run npm start

- By default application exist on localhost:5000 but you can see app path in console 

## Performance

- Work test in next browsers:
- Chrome >= 70.0
- Firefox >= 54.0
- Opera >= 58.0
